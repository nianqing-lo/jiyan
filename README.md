# 🛡️ 极验 (Geetest) 行为验证码自动化识别脚本

本项目是一个基于 Python 的自动化工具，旨在通过 Playwright 和第三方打码平台（TTShitu）自动识别并处理极验（Geetest）行为验证码（如滑块拼图）。

> **注意**：本脚本仅供学习和研究自动化测试技术使用。请勿将其用于非法用途或攻击目标网站。

## 🛠️ 功能特性

*   **自动化浏览器控制**：使用 Playwright 模拟真实用户访问极验官网演示页面。
*   **图像识别**：自动截取或获取验证码背景图，并发送至打码平台进行坐标计算。
*   **模拟人类行为**：通过模拟鼠标移动轨迹（包含随机抖动），绕过简单的行为检测。
*   **支持环境**：基于 Microsoft Edge 浏览器（Chromium内核）。

⚙️ 配置说明
在运行代码前，请务必修改以下配置项：
打码平台账号：
代码中使用了 ttshitu (超级鹰/图灵等类似平台) 的 API。你需要注册账号并充值，然后填入代码中的 username 和 password。
typeid=33：对应滑块验证码类型。
浏览器通道：
代码默认使用 channel="msedge"，请确保本机已安装 Microsoft Edge 浏览器。如果没有，可改为 channel="chrome" 或移除该参数使用 Chromium。
🚀 使用方法
将代码保存为 geetest_solver.py (或其他名称)。
修改代码中的 username 和 password 为你的打码平台凭证。
运行脚本：
bash

python geetest_solver.py
📝 代码逻辑解析
脚本主要执行以下步骤：
启动浏览器：打开极验自适应验证码演示页面。
触发验证：点击按钮唤起验证码弹窗。
获取图片：
通过正则提取背景图的 URL。
下载背景图到本地临时文件。
识别坐标：
调用 predict_base64 函数将图片发送至打码平台。
获取返回的缺口 X 轴坐标。
模拟滑动：
计算滑块起始位置。
使用 page.mouse 模拟按下、移动（带随机步长）、抬起的操作。
移动距离 = 识别坐标 - 修正值(15px)。
⚠️ 常见问题
识别失败：请检查打码平台账号余额是否充足，或 typeid 是否匹配当前的验证码类型。
元素未找到：极验可能会更新前端 class 名称（如 .geetest_bg），如果报错 Locator 找不到元素，请检查网页源码并更新选择器。
超时错误：网络波动可能导致图片下载或 API 请求超时，建议增加重试机制。
